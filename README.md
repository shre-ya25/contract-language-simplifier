Contract Language Simplifier

A web-based AI tool that simplifies complex legal or contract language into plain, easy-to-understand English using Natural Language Processing (NLP) and transformer-based AI models.

This project was developed as part of my Infosys Internship. It focuses on improving accessibility to legal and contractual information by transforming complicated text into clear and readable language.

Key Features

Simplifies complex legal and technical terms

Summarizes lengthy contract sections

Improves readability while preserving the original meaning

Uses transformer-based NLP models such as FLAN-T5 or BART

Interactive and user-friendly interface built with Streamlit

Helps users quickly understand legal content without expert knowledge

Technologies Used

Python

Streamlit

Hugging Face Transformers

Natural Language Processing (NLP)

Machine Learning

Project Objective

Legal and contractual documents often contain complex terminology and long sentences that can be difficult for many people to understand.

The objective of this project is to simplify legal and contract language using AI so that users such as clients, professionals, and non-legal individuals can easily understand the content. The system simplifies and summarizes contract text while preserving the original meaning.

This application provides a user-friendly interface where users can input contract text and receive a simplified and summarized version of the content.

How It Works

The user enters legal or contract text into the application.

The NLP model processes the text using transformer-based architectures.

The system performs text simplification and summarization.

The simplified output is displayed in clear and easy-to-understand English.

How to Run the Project

Install the required libraries

pip install streamlit transformers torch

Run the application

streamlit run app.py

Open the application in your browser and enter contract text to simplify it.
